- PRÀCTICA 1
La pràctia està dividida en diferents paquets. Cada un correspon a una part de la pràctica.
Dins de cada paquet trobem les classes que hem utilitzat pel desenvolupament d'aquesta part i també el/s main/s per executar les diferents funcionalitats implementades.
En cada part també hi trobem un paquet que conté els tests unitaris dels mètodes rellevants, així com un main per executar aquests.

IMPORTANT: El fitxer generat en el File Mail Store no s'esborra després de cada execució. Per tant, si s'executen diferents parts de la pràctica sense esborrar el fitxer pot causar errors i el mal funcionament del programa.
